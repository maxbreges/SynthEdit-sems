Value Box

Drag to change the value.

Use the Mouse Wheel to change the value in decimals. (In SynthEdit it's Alt+Mouse Wheel).

Hold Ctrl to magnify. Using the Mouse Wheel while holding the Ctrl Key changes the value in hundreds.

If just clicked, the text dialog is open.

Use Shift+Drag to change the color.

Use Ctrl+Shift+Drag to change the top brightness.

Use Double Click+Ctrl+Shift+Drag to change the bottom brightness.
