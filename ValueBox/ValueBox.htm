Value Box

Click+Drag to change the value.

If just clicked, the text dialog is open.

Use Shift+Drag to change the color.

Use the Mouse Wheel to change the value. (In SynthEdit it's Alt+Mouse Wheel).

Hold Ctrl to magnify. Using the Mouse Wheel while holding Ctrl changes the value in hundreds.
