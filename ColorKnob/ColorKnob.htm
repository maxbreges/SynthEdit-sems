Color Knob

Use SHIFT+DRAG for color change.
Use ALT+SCROLLWHEEL for subtle value change. Holding SHIFT works for color change as well.
Use CTRL+CLICK to reset the knob to the Reset Value parameter.

Pins

  • Animation Position. Range (0-1).
  • Scaled. Range determined by LOW and HIGH parameters.
  • Hex Out.
  • Mouse Down.
